# 🎬 Movie Review Detection System
A high-performance sentiment analysis application built with Streamlit and Scikit-Learn that classifies movie reviews into Positive, Negative, or Neutral categories. This system utilizes advanced natural language processing (NLP) techniques, including hybrid feature engineering and model calibration, to provide accurate and reliable predictions.

# 📊 Performance Overview
The system is optimized for high accuracy and reliability, as demonstrated in the application's performance metrics:

Model Accuracy: 90.1% achieved using the Support Vector Machine (LinearSVC) model.  

Training Speed: Optimized by training on a stratified subsample of 4,500 reviews to ensure high performance without sacrificing class representation.  

Real-time Confidence: The application provides granular probability scores for each sentiment class, ensuring transparency in every prediction.
